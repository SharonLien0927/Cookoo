# Cookoo 專案啟動說明

## 前置需求
- Node.js (建議 v18 或以上版本)
- npm (通常隨 Node.js 一起安裝)

## 啟動步驟

### 1. 安裝依賴
在專案根目錄（Cookoo 資料夾）執行：
```bash
npm install
```

### 2. 啟動開發伺服器
```bash
npm run dev
```

### 3. 開啟瀏覽器
開發伺服器啟動後，會顯示本地網址（通常是 http://localhost:3000）
在瀏覽器中開啟該網址即可看到應用程式

## 常見問題

### 白屏問題
如果瀏覽器顯示白屏，請檢查：

1. **確認依賴已安裝**
   - 檢查是否有 `node_modules` 資料夾
   - 如果沒有，執行 `npm install`

2. **檢查終端錯誤**
   - 查看執行 `npm run dev` 時是否有錯誤訊息

3. **檢查瀏覽器控制台**
   - 按 F12 打開開發者工具
   - 查看 Console 標籤的錯誤訊息
   - 查看 Network 標籤確認資源載入情況

4. **確認端口未被占用**
   - 如果 3000 端口被占用，Vite 會自動使用其他端口
   - 查看終端輸出的實際網址

### 其他指令

- **建置生產版本**：`npm run build`
- **預覽生產版本**：`npm run preview`
- **程式碼檢查**：`npm run lint`
